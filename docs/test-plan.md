## Unit Tests Required (lib/calculations.test.ts)

### calcTotal(selections)
- iPhone: multiple models, different storage tiers
- Subscriptions: year count × monthly price
- Mixed: iPhone + Mac + subscription
- Returns 0 when selections empty (defensive)

### getGrade(total)
- NT$0 → 蘋果路人
- NT$99,999 → 蘋果路人
- NT$100,000 → 蘋果愛好者
- NT$299,999 → 蘋果愛好者
- NT$300,000 → 蘋果深度使用者
- NT$600,000 → 蘋果信徒
- NT$1,000,000 → Tim Cook 最愛的人
- NT$2,000,000 → 建議你去 Apple 上班
- All 6 thresholds + boundary values

### calcAAPL(totalTWD, baseYear)
- calcAAPL(480730, 2014) → result should be > 480730 (stock grew)
- calcAAPL(480730, 2023) → smaller multiplier (fewer years)
- Invalid year fallback (year < 2007 or > today)

### getComparisons(total)
- getComparisons(50000) → only shows comparisons with trigger_min ≤ 50000
- getComparisons(5000) → only smallest trigger_min items appear
- getComparisons(600000) → all comparisons eligible

## E2E Tests (Playwright)

### Critical path: Complete flow
1. Land on `/`
2. Click 「開始計算」
3. Select iPhone 15 Pro (256GB)
4. Skip Mac, iPad, wearables categories
5. Click 「計算」
6. Verify result page shows:
   - Total amount > 0
   - Grade badge visible
   - AAPL comparison block visible
   - Share button visible

### Disabled state: No selection
1. Land on `/`
2. Click 「開始計算」
3. Do NOT select anything
4. Verify 「計算」 button is disabled (aria-disabled or cursor-not-allowed)

## Key Interactions to Verify
- Selecting multiple iPhone models (e.g., iPhone 12 + iPhone 15)
- Selecting storage variant changes price
- Count-up animation fires on result page load
- Share card download triggers Satori API call with loading state

## Edge Cases
- User selects only subscription services (no hardware year = no AAPL base year; fallback to earliest subscription year)
- Very large total (NT$3,000,000+) — grade badge + comparisons display correctly
- Very small total (NT$10,000) — only applicable comparisons shown, no Rolex

## Critical Paths
- Selection → Calculation → Receipt render: must be correct
- Grade badge threshold at exactly NT$300,000, NT$600,000, NT$1,000,000
- AAPL comparison shows "你少賺了" in correct amount
